关键词private仅能用于成员变量、成员函数。表示私有，仅有在这个类的内部才可以访问，即成员函数，构造函数，以及成员变量。
一个私有的成员变量，在别的类里，是不可访问的，比如类A里有一个 private int value = 0;
在类B里，不能 .value  

这个限制是对类的而不是对对象的，是对代码而不是对于运行的――所有的同一个类的不同对象可以相互访问各自的私有成员变量或者函数。

一个friendly的成员，可以在同一个包内被调用。

一个public的类，代表任何人都可以用这个类来定义变量；但这个类必须放在同名的.java文件中


一个包可以用另一个包的东西，在前面加上

import.包名.类名；
或者
import.包名.*； 引入包内所有的东西；

如果一个成员前面有static的定义，这个成员是属于这个类的，而不是任何对象。

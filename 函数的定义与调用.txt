public static void swap(int a, int b)
{
  int t=a;
  a=b;
  b=t;
}

public static void main(String[] args)
{
  int a=5;
  int a=6;
  swap(a,b);
}
这段代码并不能帮我把 a b 交换。
因为每个函数有自己的变量空间，参数也位于这个独立的空间中，和其他函数没有关系。


函数的每次运行，就产生一个独立的变量空间，这个空间中的变量，是这次运行独有的――本地变量
定义在函数内部的就是本地变量。

一个for循环内部定义的i，也只能在这个里面使用。
